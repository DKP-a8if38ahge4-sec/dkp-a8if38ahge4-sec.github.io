# Headline

PP Formula Condensed (Black, uppercase) for headlines; Helvetica Now Display (regular) for sub-headlines.

```jsx
<Headline size="2xlarge">Inspirierende Erlebnisse</Headline>
<Headline size="medium">Was wir tun</Headline>
<Headline subHeadline>Als Kommunikationsagentur</Headline>
<Headline size="h2" color="var(--color-white)">Auf weißem Grund</Headline>
```

## Size scale
`3xlarge` (95–200px) → `2xlarge` (75–150px) → `xlarge` (55–120px) → `large` (50–100px) → `medium` (30–60px) → `small` (20–34px) → `h1`–`h6` (semantic sizes)

All sizes are responsive — values double at 640px.

## sub-headline
Use `subHeadline` prop for category labels and section openers above a headline. Renders in Helvetica Now Display, regular weight, 14px/24px.
