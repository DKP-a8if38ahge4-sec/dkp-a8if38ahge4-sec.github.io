# Button

Pill-shaped CTA button. Use for all calls-to-action. Hover fills with solid color. Right-arrow icon on default size.

```jsx
<Button variant="red">Mehr erfahren</Button>
<Button variant="black">Jetzt bewerben</Button>
<Button variant="white">Kontakt</Button>
<Button size="large" variant="red">Alle Cases</Button>
```

## Variants
- `red` — red outline, fills red on hover. Default. Use on grey-light backgrounds.
- `black` — black outline, fills black on hover. For emphasis on light backgrounds.
- `white` — white outline, fills white with red text on hover. **Only use on red backgrounds.**

## Sizes
- `default` — standard padding (28px 61px 28px 33px), 14px font, right-arrow icon
- `large` — larger padding (38px 52px), 18px font, no arrow icon
